Web Server

A web server is software and hardware that uses HTTP (Hypertext Transfer Protocol) and other protocols to respond to client requests made over the World Wide Web

Tasks: Receives client requests, executes or refuses requested HTTP method, manages requests for static resources (file contents), manages requests for dynamic resources, replies to client requests sending proper HTTP responses, logs

Web servers content is saved in physical computers

Content in Web pages is called Web Content e.g. text, images, audio and video files

Web Client

A web browser is an application software for accessing the World Wide Web or a local website (web clients)

When a user requests a web page from a particular website, the web browser retrieves its files from a web server and then graphically renders the page on the user's screen.

A web browser is a client which requests for the data from the server: The server processes the data required in the request and provides a response to the client.
